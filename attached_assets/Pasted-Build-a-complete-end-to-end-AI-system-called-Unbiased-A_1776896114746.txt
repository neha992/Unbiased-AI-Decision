Build a complete end-to-end AI system called "Unbiased AI Decision System" that detects and fixes bias in datasets and machine learning models.

The system should follow a simple pipeline:

1. Upload Dataset (CSV)
2. Detect Bias
3. Apply Fix (basic resampling)
4. Train Model
5. Show Results (before vs after)
6. Explain decisions (feature importance or SHAP)

---

SYSTEM ARCHITECTURE:

User → Frontend UI → Backend API → ML Pipeline → Results Dashboard

---

1️⃣ FRONTEND (React or simple HTML/CSS/JS)

Include:
- File upload (CSV)
- "Analyze Bias" button
- "Auto Fix Bias" button
- Dashboard showing:
  - Bias Score
  - Graphs (distribution, comparison)
  - Before vs After metrics

---

2️⃣ BACKEND (Python with Flask or FastAPI)

Create API endpoints:
- /upload → Upload dataset
- /analyze → Detect bias
- /fix → Apply debiasing
- /train → Train model
- /report → Generate results

---

3️⃣ ML PIPELINE (Core Logic)

STEP 1: Data Ingestion
- Load dataset using pandas

STEP 2: Data Analyzer
- Detect missing values
- Detect imbalance in sensitive features (gender, age)
- Output imbalance warnings

STEP 3: Bias Detection
- Calculate:
  - Demographic parity
  - Group distribution
- Example output:
  "Male approval: 70%, Female: 40% → Bias detected"

STEP 4: Debiasing Module
- Apply resampling using sklearn
- Optionally drop sensitive features

STEP 5: Model Training
- Use Logistic Regression
- Train model on dataset

STEP 6: Evaluation
- Measure accuracy
- Calculate bias score

STEP 7: Explainability
- Show feature importance OR SHAP values
- Output insights like:
  "Income has high impact"
  "Gender may cause bias"

STEP 8: Comparison Engine
- Show before vs after table:
  Accuracy
  Bias Score

STEP 9: Report Generator
- Generate summary:
  - Bias detected
  - Fix applied
  - Final fairness score

---

REQUIREMENTS:
- Keep implementation simple and beginner-friendly
- Use clear modular structure
- Add comments in code
- Focus on clarity over complexity
- Make it suitable for hackathon demo

---

OUTPUT:
- Full backend code (Flask/FastAPI)
- Basic frontend UI
- ML pipeline scripts
- Example dataset usage
